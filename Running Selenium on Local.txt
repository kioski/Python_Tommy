Steps in running selenium automation on local.

1. Install python 2 or 3 
2. Install Selenium using pip inside python. 
   Command: [ pip install selenium ]
3. Install Sublime (Or any editor)
4. Install Latest Chrome Driver in command line using chocolatey (Make sure you are in admistrator mode)
   Command: [ choco install chocolatey ]
5. Open the Selenium test script
6. Run the code (When using Sublime use this command: Control + "B")
